# 기계 번역과 이미지 생성 멀티모달을 활용한 모델서빙 

## ✔ Overview

- <b> 배경 </b>

  OpenAPI의 Dalle2 멀티모달은 한국어 지원이 상용화되지 않은 점을 착안
  
  산문체에 강력한 기계번역은 예술성이 부여된 Dalle AI 모델과는 이질된 개념을 가짐

- <b> 해결 방안 및 개발 방향성 </b>

  text-to-image 모델에 감성어 처리가 된 기계번역을 배치하여 예술성을 높일 수 있는 모델로 배포
